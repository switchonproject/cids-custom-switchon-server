About the Published Catalogue Access API

This is the Published Catalogue Access API (OGC CSW) of the Spatial Information Platform (SIP) of the SWITCH-ON EU FP7 collaborative project.

Those are web services that provide (read only) access to the meta-data catalogue through standardised service interface mainly for the purpose to find Open Data. They can be used by any (external) client that supports the respective standard. They are currently realised as OGC Web Catalogue Service (CSW) and implemented by pyCSW. The following prototypical components are available:

  • SWITCH-ON OGC CSW (Demo)
    Endpoint of the Demo OGC CSW deployed at deltares. Provides read-only access to the Meta-Data Repository of the SIP.
  • Simple CSW Client (Third-Party Application)
    An example for an external client connected to the SIP Catalogue. The Simple CSW Client uses the CSW Endpoint listed above.
